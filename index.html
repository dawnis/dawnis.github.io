Projects

SmartFit.ai

SmartFit is a consulting project that I worked on for PM Fellow Steve Xie in Collaboration with Rohan Bhandari, an AI Fellow at Insight Data Science. In this project, I created an image-based recommender system which provides suggestions for women's tops based on an input image. It works by calculating edge information using Histogram of Oriented Gradients and color information from the HSV color values of the image in addition to using an autoencoder network (architecture here) and validated here.

Using Neural Networks to Analyze Neural Circuits

Convolutional Neural Network to Detect Ca++ Transients in Whole Brain Imaging is a project to see if convolutional layers within neural networks can learn patterns associated with neuronal firing in neural Ca++ recording data.
