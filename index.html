SmartFit.ai

SmartFit is a consulting project that I worked on for PM Fellow Steve Xie in collaboration with AI Fellow Rohan Bhandari at Insight Data Science. For this project, I created an image-based recommender system which provides suggestions for women's tops. It works by extracting image statistics such as the histogram of oriented gradients and color spectra from the HSV-converted image in addition to encoding the structure of the top in an autoencoder network (architecture here). The model was validated using the labels associated with items in the training data, the DeepFashio dataset from CUHK Media Lab.

Using Neural Networks to Analyze Neural Circuits

One type of data encountered often in neuroscience are microscopy time series in which the flashing of cells (due to a genetically encoded calcium sensitive indicator protein) corresponds to the activity of individual neurons (see video below for an example). Convolutional Neural Network to Detect Ca++ Transients in Whole Brain Imaging is a project to see if convolutional layers within neural networks can learn patterns associated with neuronal firing in neural Ca++ recording data.
